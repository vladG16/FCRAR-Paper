Summary:

This project presents a ROS-based waypoint navigation system using Clearpath's Husky A200 robot and the OptiTrack motion capture system. By using accurate external positioning data instead of traditional onboard sensors, the team streamlined the development and testing of robot navigation algorithms.

Waypoints are recorded via joystick and autonomously followed in simulation and real-world tests.
